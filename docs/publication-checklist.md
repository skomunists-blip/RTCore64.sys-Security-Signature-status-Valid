# Publication Checklist

Before pushing this repository to GitHub:

- [ ] Confirm no `*.sys` files are present.
- [ ] Confirm no `*.dmp` / memory dumps are present.
- [ ] Confirm no certificates or private keys are present.
- [ ] Confirm no risky or destructive scripts are present.
- [ ] Confirm the report does not claim unrestricted RAM read/write.
- [ ] Confirm the report does not claim GPU command execution.
- [ ] Confirm all evidence files are text-only logs.
- [ ] Confirm repository title and description clearly state defensive research.
