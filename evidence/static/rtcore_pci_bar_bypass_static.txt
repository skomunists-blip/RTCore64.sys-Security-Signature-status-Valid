===== STATIC INTERPRETATION CHECKLIST =====
1. Raw I/O write branches check exact port 0xCFC before denying writes to PCI BAR offsets.
2. PCI config data window consists of byte ports 0xCFC-0xCFF; exact-port-only checks may leave partial writes via 0xCFD/0xCFE/0xCFF.
3. Raw write branches also permit writes to 0xCF8 unless another caller-side policy blocks them; this changes PCI config address selection.
4. HalSetBusDataByOffset IOCTL 0x80002054 blocks BAR offsets 0x10-0x27, but raw port writes are a separate path.
5. This is a bypass candidate, not a confirmed RAM read/write primitive without dynamic proof.
