# Security Policy

This repository is a defensive research package.

## Scope

The repository documents a security assessment of `RTCore64.sys` and related lab observations.

## Out of Scope

The repository intentionally excludes:

- driver binaries;
- crash dumps and full memory dumps;
- exploit code;
- shellcode;
- token stealing examples;
- arbitrary physical RAM read/write tooling;
- destructive reproduction scripts.

## Reporting Issues

If you believe this repository accidentally includes sensitive or unsafe material, please open an issue or contact the maintainer privately before redistributing the content.

## Responsible Use

Use the materials only in systems you own or are authorized to test. The included tools are intended for static review or read-only telemetry collection.
